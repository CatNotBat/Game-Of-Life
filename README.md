# Game of Life

Conway's Game of Life with a tkinter GUI. Click start to seed the board with random patterns, then watch it evolve. You can pause and step through generations manually.

## How to run

```bash
python game_of_life/app.py
```

Uses tkinter so no extra dependencies needed.

## Controls

- **Start Game** — seeds the board with random patterns and runs the simulation
- **Stop** — pauses so you can inspect the board
- **Step** — advance one generation at a time (while paused)
- **Reset** — clear everything

50x50 grid, 100ms per generation.
